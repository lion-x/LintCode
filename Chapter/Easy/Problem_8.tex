\section{Problem ID: 8  Rotate String}
\subsection{Description}
Given a string and an offset, rotate string by offset. (rotate from left to right)

\subsection{Example}
Given "abcdefg".

offset=0 => "abcdefg"

offset=1 => "gabcdef"

offset=2 => "fgabcde"

offset=3 => "efgabcd"
